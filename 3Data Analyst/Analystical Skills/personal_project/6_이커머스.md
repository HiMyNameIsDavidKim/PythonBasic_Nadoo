# 코호트 분석

## `[시나리오 요약]`
* 재구매율 떨어지는 경향 원인 파악 및 대응
* 최초 구매월 기준 코호트 형성
* 코호트별 월별 재구매율, 이탈률 분석
* 코호트 인덱스(유입 이후 지난 개월 수)
* (hint: 특정 코호트에서 재구매율, 이탈률 높다)
* 학습 결과물: 재구매 패턴, 이탈 패턴, 기간별 행동 변화, 세그멘트별 행동 변화
* 추가 결과물: 기간별 행동변화(히트맵, 그래프), 마케팅 전략 제안
* 유형: 코호트 분석
