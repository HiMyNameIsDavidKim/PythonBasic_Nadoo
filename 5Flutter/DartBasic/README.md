## `[Grammar]`

### [enum]
* 상태를 열거해 묶어서 정의한다.
<br><br>

### [arrow function]
* 간소화 표현.
* => 뒤에 return을 바로 작성한다.
<br><br>

### [typedef]
* 여러 종류의 함수를 담을 수 있다.
* 내부는 arrow function 으로 작성한다.
* 실제 사용 시 파이썬의 람다와 유사하게 사용된다.
